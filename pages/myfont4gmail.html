My Font for Gmail

My Font for Gmail is a Google Chrome extension that changes the default font in Gmail to any font in your system.

The purpose for this plugin is to make you emails look better in gmail on your own computer.

This extension changes the default font of Gmail to any font you like. To use it:

  1. Click the "wrench" icon on upper-right corner of Chrome,
  2. Select tools->Extensions->My Font for Gmailâ„¢->options.
  3. Enter any font in your system.
  4. Click "Save".
  5. Refresh the Gmail page or re-open it.

Please note that this extension works best for plain-text emails. Rich-text(HTML) emails will possibly remain its original font because those emails can set their own font for each element. Usually this won't bother you since they are already well-formatted.

Source code is published at my git repository under MIT license.

Please report any bugs or suggestions to https://github.com/lancelotj/myfont4gmail/issues.
